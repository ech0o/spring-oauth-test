# oauth2 based on spring security

> 一个基于 spring security oauth2 的demo

此项目是oauth2的一个简单的实现，模式是password，token存在redis中

### 项目结构
``` 
    oauth-server 认证服务器
    eureka-server 注册服务
    oauth-client 客户端
    zuul 网关 
```
### 快速开始


1. 
``` bash
# 克隆项目到本地
git clone git@github.com:ech0o/spring-oauth-test.git 
```
2. 修改auth-server中的redis设置

3. 启动模块
